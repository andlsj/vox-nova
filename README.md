# Vox Nova

Vox Nova é um projeto editorial independente dedicado à tradução
e estudo textual.

## Projeto Neos

O Projeto Neos é uma tradução independente das Escrituras,
com base crítica e linguagem contemporânea.

Textos disponíveis:
- 2 João
- 3 João
- Filemom
- Judá
- Tito

O projeto está em desenvolvimento contínuo.
